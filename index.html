Работа в консоли

ls - вывести список содержимого папки, в которой находимся

cd - сменить папку

mkdir - создать папку

rmdir - удалить папку

rm –rf - удалить содержимую папку

touch - создаnm файл со своим расширением

code - тоже, что и touch, только файлы, которые создали, открываются

Работа с Git и GitHUB

  • git init - инициализировать репозиторий
  • git status - Показывает текущий статус репозитория
  • git add - Отслеживает изменения файлов в репозитории
  • git commit "" - Сохраняет изменения в коммит репозитория
  • git branch - Работа с ветками в репозитории
  • git config - Конфигурация и параметры git
  • git push - Заливает текущие локальные коммиты в удаленный репозиторий
